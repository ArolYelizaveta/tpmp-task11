# StudentAttestation - Task11
iOS-приложение на Swift для задачи 11: аттестация студентов, регистрация пользователя, ввод результатов экзаменов/зачетов и расчет средних показателей по группам и предметам.

## Структура проекта
StudentAttestationTask11App.swift — точка входа SwiftUI-приложения.
ContentView.swift — вкладки и все основные SwiftUI-экраны.
Models.swift — модели студента, результата сессии и средних показателей.
Storage.swift — работа с UserDefaults и .plist.
IBRegistrationViewController.swift — UIKit-контроллер.
Main.storyboard — Interface Builder файл с подключенными элементами.

## Запуск
Склонируйте проект для запуска.
Откройте StudentAttestationTask11.xcodeproj в Xcode.
Выберите iPhone Simulator.
Запустите target StudentAttestationTask11.
Основной экран откроется как SwiftUI-приложение с тремя вкладками: Студент, Сессия, Средние.
